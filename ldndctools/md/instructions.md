# LandscapeDNDC preprocessing tool
## ... a small demo app that makes creating input for LDNDC more fun 

This app is merely an interface to the LandscapeDNDC tools preprocessing CLI.
Instead of providing all relevant parameters via the CLI and the yaml config
you can specify them interactively.

**👈 To start the app use the sidebar to the left and click _RUN THE APP_**
